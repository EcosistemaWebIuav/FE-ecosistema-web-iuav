## Quick start

1.  **Install dependencies.**

    ```shell
    npm i
    ```

2.  **Start developing.**

    To start your project up run

    ```shell
    gulp init
    gulp dev
    ```

    Your site is now running at http://localhost:3000

3.  **Build (production)**

    To build minified assets ready for production run

    ```shell
    gulp dist
    ```

    All files will be created under  `dist` directory.
